# AI Sentiment Analysis Platform

A professional NLP-based web application that analyzes text sentiment
using modern Natural Language Processing techniques.

---

## Features

- Real-time sentiment analysis
- Positive / Neutral / Negative classification
- Confidence score visualization
- REST API support
- Rate limiting for security
- Modern dark UI

---

## Tech Stack

- Python
- Flask
- NLP (VADER & Transformers)
- HTML / CSS / JavaScript
- Chart.js (visual analytics)

---
